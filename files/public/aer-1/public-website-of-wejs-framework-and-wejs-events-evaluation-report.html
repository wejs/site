Accessibility Evaluation Report for We.js and We.js Events

Report Creator: Talita Pagani, November 11, 2015

Evaluation Commissioner: Alberto Souza - Head of We.js

Summary of the evaluation findings

This report describes the conformance of the We.js and We.js Events Web sites with W3C's Web Content Accessibility Guidelines (WCAG) 2.0. The review process is based on the W3C's Conformance Evaluation.

Conformance evaluation of Web accessibility requires a combination of semi-automated evaluation tools and manual evaluation by an experienced reviewer. The evaluation results in this report are based on evaluation conducted on the following date(s): October 27-28-29-30 and November 10-11 (2015). The Web site may have changed since that time.

Based on this evaluation, the We.js and We.js Events Web site is close to meeting WCAG 2.0, Conformance Level AA. Detailed review results are available below. Resources for follow-up study are listed at the end of this document. Feedback on this evaluation is welcome.

Frequently occurring issues and patterns observed include low contrast between text and background, missing focus indicator for elements and form controls, missing ARIA roles/states/attributes to improve content readability for screen readers and low performance of client-side scripts. Thus, these issues are high priority to be solved.

Scope of the evaluation

Website namePublic Website of We.js Framework and We.js Events
Scope of the website

All web content of the public website of We.js Framework and We.js Events located at http://wejs.org/ and http://events.wejs.org/, respectivelly. This evaluation also include restricted area of both websites, accessed via user login.

Conformance targetLevel AA
Additional evaluation requirements

The report will include a list of all errors identified by the evaluator, as well a description of the problem and proposals of improvement for both websites.

Accessibility support baseline

Both websites were tested using Google Chrome v. 46.0.2490.80 m with NVDA and Firefox 42.0 with NVDA.
